## AgriHouse Project Setup Complete

This is a React + Vite + Tailwind CSS project initialized with the following configuration:

### Stack
- React 19.2.4
- Vite 8.0.4
- Tailwind CSS 4.2.2 with @tailwindcss/vite plugin
- React Router 7.14.1
- ESLint 9.39.4

### Development

- **Start dev server**: `npm run dev`
- **Build**: `npm run build`
- **Preview build**: `npm run preview`
- **Lint code**: `npm run lint`

### Key Files
- `vite.config.js` - Vite configuration with React and Tailwind plugins
- `src/index.css` - Global styles with Tailwind CSS directives
- `src/App.jsx` - Main React component
- `.eslintrc.js` - ESLint configuration

### Features
- Hot Module Replacement (HMR) for instant updates during development
- Tailwind CSS utility classes for styling
- React Router for client-side routing
- Code linting with ESLint

Dev server running at: http://localhost:5173/
